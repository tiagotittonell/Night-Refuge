## NOTAS DE USO
## ═══════════════════════════════════════════
# - Resolución de iconos: 128x128 o 64x64 según uso
# - Resolución de fondos de panel: 1920x1080 o 960x540
# - Formato: PNG con transparencia donde se indique
# - Importar en Unity como Sprite (2D and UI)
# - Carpeta sugerida: Assets/Sprites/UI/ o Assets/Sprites/Panels/
# - Filter Mode: Bilinear
# - Compression: None o Low Quality para mantener nitidez en UI
